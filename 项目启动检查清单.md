## 🆘 常见问题解决

### 编译问题
**Q: CMake找不到vcpkg包？**
```bash
# 确保使用正确的工具链文件
cmake .. -DCMAKE_TOOLCHAIN_FILE=路径/vcpkg/scripts/buildsystems/vcpkg.cmake
```

**Q: 编译器不支持C++17？**
```bash
# 检查编译器版本
g++ --version
# 或者在CMake中指定编译器
cmake .. -DCMAKE_CXX_COMPILER=g++-8
```

### 网络连接问题
**Q: WebSocket连接失败？**
1. 确认后端服务器已启动
2. 检查端口8080是否被占用
3. 暂时关闭防火墙测试

### 前端问题
**Q: 页面样式不正确？**
1. 检查CSS文件路径
2. 确认字体文件加载
3. 使用浏览器开发者工具调试
